## 목적

python class, typing 활용.

## 목표

python을 사용해서 하스스톤 CLI를 구현한다.

1차 구현

- 제이나만 사용, 공용캐, 제이나 캐, 제이나 스킬들
- 보스잡기 레이드 - 기존 하스스톤 퀘스트로 고정
- 키보드 입력으로 보스를 잡는다.

...

## 동작

NPC 대전해서 잡기 - e.g. 체력 100, 도끼든 광전사 사냥

### CLI 표현 방식

12시, 6시

```
플레이어(공, 체)

1. 유닛(공,체,특수기) | ... | 7.유닛 (공,체,특수기)

---------------
1. 유닛 | ...| 7. 유닛

(공)플레이어(체)
```

## 유저 시나리오

유저 인풋은 숫자키 입력
